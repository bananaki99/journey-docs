JOURNEY-DOCS
============

:mod:`Journey` is a platform primarily aimed to host and exchange various kind of data collected or produced by International Organisations, Central Banks, Statistical Offices and/or Supervision Offices.

Section :doc:`domain` describes the various kind of data that :mod:`Journey` can handle.

Section :doc:`id_type` describes the format types that is used to identify
various kind of units in :mod:`Journey`. 

Section :doc:`sid` provides the names and the identification codes of the
reporting agents.
